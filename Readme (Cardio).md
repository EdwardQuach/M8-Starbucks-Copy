Heart Disease Prediction Dataset



This repository contains a subset of the Heart Disease Data. Originating from different databases, This dataset has been widely used since its creation in 1988 for predictive modeling.



Dataset Overview



&nbsp;   Source: The dataset is a compilation from four separate medical centers: Cleveland Clinic Foundation, Hungarian Institute of Cardiology, University Hospital (Zurich), and V.A. Medical Center (Long Beach).



&nbsp;   Attributes: While the original dataset contains 76 attributes, all published experiments and this project focus on a subset of 14 key features. Patient identifiers, such as names and social security numbers, have been replaced with dummy values for privacy.



&nbsp;   Target Variable: The target field is the binary class variable indicating the presence of heart disease.



&nbsp;       0: No heart disease



&nbsp;       1: Presence of heart disease



Attribute Information



The following is a list of the 14 attributes used in the dataset:



&nbsp;   age: Age of the patient.



&nbsp;   sex: Gender of the patient.



&nbsp;   cp: Chest pain type (4 categories).



&nbsp;   trestbps: Resting blood pressure (in mm/Hg).



&nbsp;   chol: Serum cholesterol (in mg/dl).



&nbsp;   fbs: Fasting blood sugar (> 120 mg/dl).



&nbsp;   restecg: Resting electrocardiographic results (3 categories).



&nbsp;   thalach: Maximum heart rate achieved.



&nbsp;   exang: Exercise-induced angina.



&nbsp;   oldpeak: ST depression induced by exercise relative to rest.



&nbsp;   slope: The slope of the peak exercise ST segment.



&nbsp;   ca: Number of major vessels (0-3) colored by fluoroscopy.



&nbsp;   thal: A blood disorder called Thalassemia (3 categories: normal, fixed defect, reversible defect).
